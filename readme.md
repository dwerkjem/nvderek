# nvDerek

nvDerek is a simple, lightweight, neoVim config that is easy to install and use. It is designed to be a starting point for users who are new to neoVim or who want a simple, easy-to-use config.

## Installation

To install nvDerek, simply clone the repository into your `~/.config/nvim` directory:

```bash
git clone https://github.com/dwerkjem/nvderek.git ~/.config/nvim
```

Then, open `nvim` and close it. It will install the plugins automatically and install the necessary language servers. You may need to restart `nvim` a few times to get everything set up.

## Features

nvDerek includes the following features:

- lazy-loading

- auto-completion

- syntax highlighting

- linting

- code formatting

- git integration

- fuzzy file finding

- and more!
